### 新增分类：山西革命

**分类背景**：
山西是革命老区，是八路军抗日的主战场之一。山西也是中医药的重要发源地，拥有丰富的中药材资源，历史上被誉为"北方药都"。本分类将山西的革命历史与丰富的草药资源相结合，展现山西人民在革命战争年代的贡献。

**题目特色**：

- ✅ 结合山西地理特点和药材资源
- ✅ 融入八路军在山西的抗战历史
- ✅ 介绍晋商与中药贸易的历史
- ✅ 展现山西中医名家的贡献

**代表题目**：

1. 平型关大捷与草药止血
2. 百团大战与太行山药材

4. 傅山先生与山西中医
5. 晋商茶马古道与甘草贸易

## 🎨 山西分类题目示例

### 1. 历史文化类

```
问题：山西是中医药的重要发源地之一，被誉为"北方药都"的是？
答案：运城
解析：运城古称河东，药材资源丰富，历史上就是北方重要的药材集散地。
```

### 2. 革命历史类

```
问题：平型关大捷后，八路军用山西当地哪种草药为伤员止血？
答案：大蓟
解析：大蓟凉血止血，太行山区常见。平型关大捷是抗战以来首次大捷。
```

### 3. 名人医学类

```
问题：山西中医名家傅山，善用哪种草药组成补气名方？
答案：黄芪
解析：傅山创制的"黄芪当归汤"，补气养血，医德医术皆为后世楷模。
```

### 4. 地方特产类

```
问题：山西雁门关一带生长的哪种草药，有"小人参"之称？
答案：党参
解析：山西党参素有"小人参"美誉，补中益气效果显著。
```
