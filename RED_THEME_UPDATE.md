# 🚩 红色主题更新说明

## 📝 更新概述

本次更新将原有的纯草药问答系统升级为**红色草药问答系统**，将中国共产党的红色革命精神与传统中医草药知识深度融合，打造一个兼具教育意义和文化价值的学习平台。

## 🎯 更新内容

### 1. 题目内容全面更新

将所有题目改为**红色精神 + 草药知识**结合的形式：

#### 题目特点

- ✅ 每道题目关联真实的革命历史事件
- ✅ 融入草药在革命战争年代的实际应用
- ✅ 解析部分包含历史背景和草药知识

#### 示例对比

**更新前（普通草药题）：**

```
问题：人参的主要功效是什么？
答案：大补元气
解析：人参具有大补元气、复脉固脱的功效。
```

**更新后（红色草药题）：**

```
问题：爬雪山过草地时，红军用哪种药材帮助战士抵御寒冷、补充体力？
答案：人参
解析：人参大补元气，红军在极端恶劣环境下用野山参救治体力不支的战士，
      展现了革命乐观主义精神。
```

## 🔧 如何应用更新

### 方案一：重新初始化数据库（推荐用于新项目）

```bash
# 1. 删除旧数据库
mysql -u root -p -e "DROP DATABASE IF EXISTS caoyao_qa;"

# 2. 使用新的初始化脚本
mysql -u root -p < database/init.sql
```

### 方案二：更新现有数据库（保留用户数据）

```bash
# 使用更新脚本（会清空题目和答题记录，保留用户数据）
mysql -u root -p < database/update_to_red_theme.sql
```

⚠️ **注意**：更新脚本会：

- ✅ 保留 `users` 表中的用户数据
- ✅ 保留 `user_statistics` 表结构
- ⚠️ 清空 `questions` 表并插入新题目
- ⚠️ 清空 `answer_records` 表
- ⚠️ 重置所有用户的统计数据为 0

### 方案三：仅查看新题目

如果只想查看新题目内容，可以直接查看：

- `database/init.sql` 中的 INSERT 语句
- `database/update_to_red_theme.sql`
